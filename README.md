# Tableau de Bord d'Analyse des Magasins Asswak Assalam Analytics

Ce projet est un tableau de bord d'analyse pour la gestion des magasins, permettant d'effectuer des analyses statistiques sur les chiffres d'affaires, les surfaces et les effectifs des magasins.

## Installation et Utilisation

### Étape 1 : Cloner le Dépôt

```bash
git clone https://github.com/kadiriayman96/Asswak-Assalam
```

### Étape 2 : Naviguer vers le Répertoire du Projet

```bash
cd Asswak-Assalam
```

### Étape 3 : Ouvrir l'Application

1. Trouvez le fichier `index.html` dans le répertoire du projet.
2. Ouvrez `index.html` dans votre navigateur web préféré (Google Chrome, Firefox, etc.).

### Étape 4 : Connexion en tant qu'Admin

- Après avoir ouvert `index.html`, vous serez redirigé vers la page de connexion pour l'admin.

### Étape 5 : Accéder au Tableau de Bord

- Après une connexion réussie, vous serez dirigé vers l'interface principale du tableau de bord.

### Étape 6 : Navigation entre les Pages

- **Dashboard Analytics :** Cette page affiche des métriques analytiques et des graphiques basés sur les paramètres sélectionnés (année, métrique).
  
- **Liste des Magasins :** Vous pouvez voir une liste de tous les magasins sous forme de cartes (verticale ou horizontale).
  
- **Ajouter Magasin :** Ajoutez un nouveau magasin avec des détails tels que le CA, l'année, la surface et les effectifs.
  
- **Détails du Magasin :** Opérations CRUD pour les détails du magasin, y compris le CA, l'année, la surface et les effectifs.

### Étape 7 : Analyser les Métriques

- **Sélectionner une Année :** Utilisez le menu déroulant pour sélectionner une année spécifique pour les analyses.
  
- **Visualiser les Métriques Statistiques :** Le tableau de bord affiche la moyenne arithmétique pondérée, le mode, la médiane, l'écart-type et le coefficient de variation pour l'année sélectionnée.

### Étape 8 : Interagir avec les Graphiques

- **Graphique des Plages de CA :** Visualisez la fréquence des magasins catégorisés par plage de CA.
  
- **Graphique de Distribution des Surfaces :** Explorez la distribution des surfaces des magasins.
  
- **Graphique de Distribution des Effectifs :** Visualisez la répartition des effectifs (nombre de personnel) dans les magasins.

### Étape 9 : Déconnexion

- Cliquez sur le bouton "Déconnexion" pour vous déconnecter en toute sécurité de l'application.
